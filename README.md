# subirArchivos

## Descripción

Este es un ejemplo donde se muestra como subir archivos utilizando lenguaje PHP y como mostrar esos archivos que están cargados en un directorio

## Preparación

- Servidor web apache
- PHP
- Permisos de escritura en carpeta de archivos
